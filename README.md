# 💸 Money Converter

A simple and clean currency converter built with HTML, CSS and JavaScript. 🌸

## ✨ Features

- 🌍 Converts between 75+ currencies with country flags
- 📡 Live exchange rates via ExchangeRate-API
- 🔄 Falls back to approximate rates if offline
- 🔃 Swap button to quickly flip From and To currencies

## 🙂 How to Use

1. 🔢 Enter the amount you want to convert
2. 🏳️ Select the **From** currency
3. 🏳️ Select the **To** currency
4. 💗 Click **Convert**

## 📶 Live Rates

Rates are fetched automatically from [ExchangeRate-API](https://www.exchangerate-api.com) on load. The status below the title will confirm if live rates are active.
